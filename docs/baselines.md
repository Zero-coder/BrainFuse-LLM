# Baselines and Related Repositories

BrainFuse-LLM is compared with both subject-specific and subject-agnostic fMRI decoding approaches.

## Captioning Baselines

- SDRecon
- BrainCap
- BrainChat
- UniBrain
- UMBRAE

## fMRI-QA Baselines

- OneLLM
- UMBRAE
- BrainChat
- MindBridge
- UniBrain

## Documentation References

The public repository organization follows common conventions used by recent brain-decoding repositories:

- Installation instructions with a conda environment.
- Dataset section describing NSD and third-party annotations.
- Checkpoint section for future model-weight release.
- Usage examples for pretraining, finetuning, and evaluation.

No code is copied from related repositories.
